# 🔥 Brazil Forest Fire Classification  



🚀 A Machine Learning project to classify forest fire intensity in Brazil using historical data.

---

## 📸 Project Preview
![Output](output.png)

---

## 📌 Project Overview
This project analyzes forest fire data from Brazil and builds a classification model to predict fire intensity levels.

---

## 📂 Dataset
- Amazon Forest Fire Dataset  
- Features:
  - Year  
  - State  
  - Month  
  - Number of Fires  
  - Date  

---

## 🧹 Data Preprocessing
- Converted date column to datetime  
- Extracted day feature  
- Handled missing values  
- Encoded categorical variables  

---

## 🎯 Problem Statement
Classify fire intensity into:
- 🔵 No Fire  
- 🟢 Low  
- 🟡 Medium  
- 🔴 High  

---

## 🤖 Models Used
- Decision Tree Classifier  
- Random Forest Classifier  

---

## 📊 Evaluation Metrics
- Accuracy Score  
- Confusion Matrix  
- Classification Report  

---

## 📈 Results
- Achieved good accuracy in predicting fire intensity  
- Random Forest performed better than Decision Tree  

---

## ⚙️ Tech Stack
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Scikit-learn
